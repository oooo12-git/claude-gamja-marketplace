# Threads → Instagram 변환 가이드

## 플랫폼 차이점

| 항목 | Threads | Instagram |
|------|---------|-----------|
| **링크** | 클릭 가능 | 클릭 불가 → "프로필 링크에서 확인" |
| **핸들** | @jeongcheogi.gamza | @jeongcheogi_gamja (언더스코어) |
| **해시태그** | 최소 사용 | 최대 5개 (인스타그램 제한) |
| **길이** | 여러 thread로 분산 | 한 caption에 집약 |
| **톤** | 반말 + 대화체 | 존댓말 + 대화체 (기존 콘텐츠 참고) |

## 변환 체크리스트

- [ ] URL → "프로필 링크에서 확인해보세요"
- [ ] @ 핸들 → 언더스코어 형식 확인
- [ ] 계산/가정의 근거를 수식처럼 명시
- [ ] 추상적 표현 → 구체적 수치/시점 추가
- [ ] 1인칭 시점("저는", "제가") 검토
- [ ] 해시태그 최대 5개 추가 (인스타그램 제한)
- [ ] 어투 통일: 기존 Instagram 콘텐츠 참고하여 어투 맞추기
- [ ] 파일 저장: `social/instagram/contents/` 에 저장

## 어투 통일

**중요**: 변환 전 기존 Instagram 콘텐츠(`social/instagram/contents/*.md`)를 읽어서 어투를 맞춰야 합니다.

### 어투 변환 예시

| Threads (반말) | Instagram (존댓말) |
|----------------|-------------------|
| 정리해둘게 | 정리해뒀습니다 |
| 확인해! | 확인해두세요! |
| 마감됨 | 마감됩니다 |
| 추천해 | 추천드려요 |
| 정해두기 | 정해두세요 |
| 등록해뒀어 | 등록해뒀습니다 |
| 확인해봐 | 확인해보세요 |

### 어투 특징
- "~했습니다", "~하더라고요", "~해보세요" 형태
- "저는" 1인칭 사용 가능
- 질문형 마무리 ("~아닐까요?")

## 파일 저장

변환 완료 후 `social/instagram/contents/` 디렉토리에 저장합니다.

### 파일명 규칙
- 원본 Threads 파일명을 기반으로 생성
- 형식: `{콘텐츠명}_{YYYYMMDD}.md`
- 예시: `jeongcheogi-application_20260105.md`

### 파일 구조
```markdown
# instagram feed

## image
### image describe prompt
[이미지 묘사 프롬프트]

### overlay text
[이미지 위에 들어갈 텍스트]

## caption
[변환된 Instagram caption]
```

## Caption 구조

```
[킥 문장] + 이모지

[배경/맥락 설명 - 2~3문장]

[이모지] 섹션 제목
▪️ 데이터 포인트 1
▪️ 데이터 포인트 2

[이모지] 섹션 제목
▪️ 근거 = 결론: 수치
▪️ 계산식 = 결론: 수치

[인사이트/결론 - 1~2문장]

[이모지] CTA
[프로필 링크 안내]

[이모지] 계정 태그 @계정명

#해시태그1 #해시태그2 ...
```

## 이모지 사용법

### 섹션 구분용
- 📊 데이터/통계
- 📈 증가/트렌드
- 💡 인사이트/아이디어
- 📚 학습/정보
- ✨ CTA/마무리
- 📌 핵심 포인트
- 🚀 시작/런칭

### Bullet용
- ▪️ 일반 항목

## 변환 예시

### Before (Threads)
```
그런데 정처기 감자 DAU를 뜯어보니까 이유가 뭔지 알겠더라구.

평소 DAU: 800명대
시험 전날 DAU: 3,000명

단순히 이걸
꾸준히 공부한 사람 = 800명 (27%)
벼락치기 장인 = 2,200명 (73%)
이렇게 가정해보니까

합격률 23.9%랑 꾸준히 공부한 사람의 비율이 거의 일치하지?
```

### After (Instagram)
```
📈 DAU 데이터가 말해주는 것
▪️ 평소 DAU: 800명대
▪️ 시험 전날 DAU: 3,000명

💡 단순 가정
▪️ 평소 DAU = 꾸준히 공부한 사람: 800명 (27%)
▪️ 시험 전날 DAU - 평소 DAU = 벼락치기 장인: 2,200명 (73%)

합격률 23.9%와 꾸준히 공부한 사람 27% 비율이 거의 일치합니다.
```

### 변환 포인트
1. **구조화**: 이모지로 섹션 구분, bullet으로 정리
2. **근거 명시**: "단순히 이걸" → "단순 가정" + 수식 형태로 근거 표시
3. **수치 구체화**: "비율" → "27% 비율"

## 해시태그 작성 팁

**인스타그램은 해시태그를 최대 5개까지만 허용합니다.** 그 이상 넣으면 게시가 차단됩니다.

### 구성 (5개 내에서)
- 대형 태그 (100만+): 1~2개 (노출용)
- 중형 태그 (10만~100만): 2~3개 (경쟁 균형)
- 니치 태그 (1만 이하): 1개 (타겟팅)

### 정처기 관련 해시태그 예시
```
#정보처리기사 #정처기실기 #자격증공부 #IT자격증 #정처기공부
```
