## 공통 패턴

### 1. 킥 문장
- 숫자 + 결과 조합 ("23.9%만 합격", "150만 개 발견")
- 의외성/반전 ("나이키테크 검색량 급증")

### 2. 섹션 구분
- 이모지 + 굵은 제목
- 📌📊✅💡🚀 등 목적에 맞는 이모지

### 3. Bullet 스타일
- ▪️ 가장 깔끔
- • 도 사용 가능

### 4. 마무리
- CTA + 계정 태그
- 해시태그 6~10개
